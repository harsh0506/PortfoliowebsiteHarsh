QuizzyBrains

Engaging Quiz App: Test your knowledge, manage questions. AWS backend, React UI, user-friendly experience for interactive learning and administration.

The Quiz App is a serverless quiz application built using a combination of AWS services on the backend and modern web technologies on the frontend. The application allows users to take quizzes and administrators to manage quiz questions and messages from the contact us page. It leverages the power of AWS Lambda, API Gateway, DynamoDB, React, and more to create a seamless and efficient quiz-taking experience.

  • React Frontend : Interactive user interfaces, smooth navigation, and engaging learning experience.
  • Tailwind CSS : Modern styling, responsive design, and visually appealing user interface.
  • AWS Serverless : Efficient backend, scalable architecture, and seamless application performance.
  • IndexedDB : Fast data caching, offline access, and improved user interaction responsiveness.
